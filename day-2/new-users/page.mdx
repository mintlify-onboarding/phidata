---
title: Setup workspace for new users
---

New users should follow these steps to setup an existing workspace:

1. Clone the git repo and `cd` into the workspace directory

<CodeGroup>

```bash Mac
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

```bash Windows
git clone https://github.com/[YOUR_GIT_REPO].git

cd your_workspace_directory
```

</CodeGroup>

2. Create and activate a virtual env

<CodeGroup>

```bash Mac
python3 -m venv aienv
source aienv/bin/activate
```

```bash Windows
python3 -m venv aienv
aienv/scripts/activate
```

</CodeGroup>

3. Install `phidata`

<CodeGroup>

```bash Mac
pip install -U phidata
```

```bash Windows
pip install -U phidata
```

</CodeGroup>

4. Setup workspace

<CodeGroup>

```bash Mac
phi ws setup
```

```bash Windows
phi ws setup
```

</CodeGroup>

5. Copy `workspace/example_secrets` to `workspace/secrets`:

<CodeGroup>

```bash Mac
cp -r workspace/example_secrets workspace/secrets
```

```bash Windows
cp -r workspace/example_secrets workspace/secrets
```

</CodeGroup>

6. Start workspace:

<Note>

Install <LinkOpenNewTab href="https://docs.docker.com/desktop/install/mac-install/" text="docker desktop" /> to run this app locally

</Note>

<CodeGroup>

```bash Terminal
phi ws up
```

```bash With Options
phi ws up --env dev --infra docker
```

```bash Shorthand
phi ws up dev:docker
```

</CodeGroup>

7. Stop workspace:

<CodeGroup>

```bash Terminal
phi ws down
```

```bash With Options
phi ws down --env dev --infra docker
```

```bash Shorthand
phi ws down dev:docker
```

</CodeGroup>
